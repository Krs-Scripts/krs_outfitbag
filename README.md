# krs_outfitbag

**Simple OutfitBag**

**Resmon  0.00 ms**

**Possibility to choose the fivem-appearance or the illenium appearance**

* ox_inventory/data/items.lua

```lua

	['outfitbag'] = {
		label = 'Outfitbag',
		consume = 0,
		weight = 1,
		client = {
			export = 'krs_outfitbag.outfitbag'
		}
	},
```
